# ابدأ من هنا — Techno Minds V60.5.0

هذه نسخة المصدر الكاملة لمنصة Techno Minds للبرمجة والذكاء الاصطناعي، ومضبوطة على مشروع Firebase:

```text
eng-amr-khaled-academy
```

## الموجود في النسخة

- موقع عربي متجاوب بهوية Techno Minds ومحتوى برمجة.
- محاضرات وأسئلة مؤمّنة بالكود ومفلترة تلقائيًا حسب مسار الطالب.
- خانات الأكواد والهواتف والدرجات تقبل أرقامًا فقط.
- حجوزات ومجموعات وطلاب وبوابة طالب وولي أمر.
- لوحة إدارة للطلاب والحضور والدرجات والمدفوعات والمحتوى والاختبارات.
- هيدر إدارة سريع لتنبيهات الحجز وحفظ التغييرات ومعاينة الموقع.
- خزنة دفع بأسعار مستقلة لكل صف، وإجمالي تلقائي للمحصل والمتوقع والمتبقي.
- إصلاح دخول المالك وتفعيل حساب الإدارة من الخادم.
- صفحة عملي عامة بدون كود طالب لتشغيل Python وJavaScript وTypeScript وC وC++ وJava وC# وGo وPHP.
- حجز متدرج داخل صندوق واحد، بدون اشتراط مجموعة، ثم نقل الطالب إلى المجموعة المناسبة من لوحة الإدارة.
- صفحتا المسار التعليمي الكامل وعن المهندس عمرو خالد والشهادات.
- Firebase Functions وFirestore Rules وIndexes وStorage Rules وHosting.
- سكربت Windows آمن من مشكلة ترميز الأحرف العربية القديمة.

## تشغيل محلي سريع

```powershell
npm run dev
```

ثم افتح `http://localhost:5173`.

## النشر

راجع `README_DEPLOY.md` أو شغّل:

```powershell
.\DEPLOY-WINDOWS.cmd
```

استخدم Node.js 22 LTS. لا تضع مفاتيح سرية داخل ملفات المشروع؛ ملف `functions/.env.example` يوضح الإعدادات الاختيارية لخدمة تشغيل الأكواد.
