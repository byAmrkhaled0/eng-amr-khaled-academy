# TechnoMinds Academy

نسخة محدثة: صفحة العملي أصبحت مخصصة للحصة وفيها Playground.py بشكل أقرب لـ VS Code، والهيدر ثابت في كل الصفحات، والاختبارات تحتوي على مساحة كتابة كود للطلاب.


## Data policy
كل المحتوى الظاهر في الموقع أو بوابة الطالب يُدار من لوحة الأدمن ويتخزن في Firebase. لا توجد صفحات كورسات/محاضرات منفصلة؛ كل المحتوى التعليمي يُحفظ في materials حسب النوع.
