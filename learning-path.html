01
Programming Mindset

التفكير المنطقي وحل المشكلات

تفكيك المشكلة، ترتيب الخطوات، الخوارزميات البسيطة، وفهم كيف ينفّذ الكمبيوتر التعليمات.

  • خوارزميات بدون تعقيد
  • Flowcharts وPseudo code
  • تحديات تفكير قصيرة
02
Python Foundations

أساسيات Python

المتغيرات، الشروط، الحلقات، الدوال، القوائم، والتعامل الصحيح مع الأخطاء.

  • برامج Console عملية
  • تمارين بعد كل مفهوم
  • مشروع نهاية المستوى
03
Problem Solving

حل المشكلات بالكود

تحويل الأفكار إلى حلول قابلة للاختبار وكتابة كود أنظف وأسرع خطوة بخطوة.

  • Debugging واكتشاف الأخطاء
  • هياكل بيانات مناسبة للعمر
  • تحديات زمنية وتعاونية
04
Web Development

أساسيات الويب

بناء صفحات حقيقية باستخدام HTML وCSS وJavaScript وفهم تجربة الهاتف أولًا.

  • موقع شخصي متجاوب
  • تفاعل ونماذج وبيانات
  • رفع المشروع على الإنترنت
05
AI Concepts

مقدمة الذكاء الاصطناعي

فهم البيانات والتعلّم الآلي والرؤية الحاسوبية بطريقة مبسطة ومسؤولة.

  • تجارب بيانات مصغرة
  • استخدام أدوات AI بوعي
  • مشروع ذكي مبسط
06
Portfolio Project

مشروع التخرج والعرض

يختار الطالب مشكلة، يبني الحل، يختبره، ويوثقه ثم يقدمه كمشروع يمكن عرضه.

  • تخطيط وتنفيذ ومراجعة
  • Git ونسخ المشروع
  • عرض المشروع بثقة
