Roadmap

مراحل المسار

كل مرحلة لها هدف واضح، تدريب عملي، ومتابعة تساعد الطالب يعرف هو وصل لفين.

01

Python Basics

فهم معنى البرنامج، الأوامر، الطباعة، الإدخال، المتغيرات، وطريقة قراءة الـ Output.

Practice: print / input / variables
02

Problem Solving

تحويل السؤال إلى خطوات، استخدام الشروط والتكرار، وتجربة أكثر من حل لنفس الفكرة.

Practice: if / loops / lists
03

Web Basics

مقدمة عملية في HTML و CSS وفكرة بناء صفحة منظمة مناسبة لشكل مواقع الويب.

Practice: structure / styling
04

AI Concepts

فهم مبسط لفكرة البيانات، التدريب، التوقعات، واستخدام الذكاء الاصطناعي في مشاريع حقيقية.

Practice: data / model thinking
05

Mini Project

تنفيذ مشروع صغير مثل Calculator أو Quiz أو Student Analyzer عشان الطالب يطبق اللي اتعلمه.

Practice: build / test / present
